d-grid-features-for-bootstrap

This library has the purpose of being able to use the grid features within bootstrap without generating a css of more than 20,000 lines of code and that is responsive with the values that you want to give to each property.

Esta librería tiene el propósito de poder usar las características de grid dentro de bootstrap sin que se genere un css de más de 20,000 lineas de código y que sea responsivo con los valores que le quieras dar a cada propiedad.
